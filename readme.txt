
COMMANDS TO RUN:

1. To run live server: "live-server"
2. To detect change in file and compile sass to css: "npm run compile:sass"


Note: 
1. To initiate npm: "npm init".
2. To install sass-package: "npm install node-sass --save-dev".
3. Live server is globally installed or if we want to install live-server than run command: "npm i live-server -g".
